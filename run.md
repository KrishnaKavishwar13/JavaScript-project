# How to Run

This project is a static website containing HTML, CSS, and JavaScript. You can run it using a simple static file server.

## Prerequisites

- **Python 3.x** (pre-installed on most systems)

## Steps to Run

1. **Open Terminal**: Open your command line interface (Command Prompt, PowerShell, or Terminal).

2. **Navigate to Project**: Change into the project directory.
   ```bash
   cd d:\memory
   ```

3. **Start Server**: Run the following command to start a local web server.
   ```bash
   python -m http.server 8080
   ```

4. **Access Application**: Open your web browser and navigate to:
   [http://localhost:8080](http://localhost:8080)

## Alternative

If you have VS Code with the "Live Server" extension, you can also right-click `index.html` and choose "Open with Live Server".
